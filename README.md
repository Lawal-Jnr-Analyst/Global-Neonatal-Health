# 🌍 Global Neonatal Health Dashboard (2019–2024)

## 📊 Overview
This project presents a 6-year analysis of global neonatal health using UNICEF data. It focuses on Neonatal Mortality Rate (NMR) trends, regional disparities, and country-level performance.

The dashboard is designed to highlight where progress is being made and where urgent intervention is still needed, especially in high-burden countries.

## 🎯 Objectives
- Track global Neonatal Mortality Rate (NMR) trends from 2019 to 2024
- Identify countries with the highest neonatal mortality burden
- Compare regional performance over time
- Classify countries based on SDG-aligned risk bands
- Provide an interactive tool for public health insights

## 🛠 Tools Used
Power BI – Dashboard development
Power Query – Data cleaning and transformation
DAX – Measures and KPIs

## 📈 Key Insights
- Neonatal mortality declined steadily from 12.54 in 2019 to 11.38 in 2024, showing gradual global progress
- The rate of improvement is relatively slow, indicating that stronger interventions are still needed
- Significant regional disparities persist, with some regions improving faster than others
- High-burden regions, especially in parts of Africa and South Asia, continue to drive global mortality rates
- Low-income regions consistently record higher NMR compared to global averages
- Countries like South Sudan, Nigeria, Pakistan, Somalia, and Afghanistan remain the most affected
- A considerable number of countries still fall within moderate to crisis SDG risk bands
- Overall progress is uneven, highlighting the need for targeted, region-specific public health strategies

## 🧹 Data Cleaning & Transformation
- Removed missing and inconsistent records
- Standardized country names for mapping
- Created calculated measures (Avg NMR, YoY Change, Risk Bands)
- Validated aggregation logic for totals and averages
